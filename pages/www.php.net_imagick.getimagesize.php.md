# Imagick::getImageSize



Practical use to get the dimensions of the image:<br><br>

```
<?php
$image = new Imagick($image_src);
$d = $image-&gt;getImageGeometry();
$w = $d[&apos;width&apos;];
$h = $d[&apos;height&apos;];
?>
```
